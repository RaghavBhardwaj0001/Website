# Global Trade Exchange

A modern, responsive website for a global import-export business platform that connects verified buyers and sellers worldwide.

## 🌐 Live Preview

Open `index.html` in your browser or deploy to GitHub Pages.

## 📁 Project Structure

```
├── index.html          # Home / Landing page
├── about.html          # About Us
├── services.html       # Services
├── membership.html     # Membership Plans & Pricing
├── contact.html        # Contact Us
├── dashboard.html      # User Dashboard Demo
├── css/
│   └── styles.css      # Global styles & design system
├── js/
│   └── main.js         # Interactive features
└── README.md
```

## ✨ Features

- **Dark + Gold Premium Theme** — Professional business design
- **6 Complete Pages** — Home, About, Services, Plans, Contact, Dashboard
- **Login & Register Modal** — User authentication UI
- **Verified Leads Database** — Dummy buyer/seller listings
- **Commission System UI** — Dashboard with earnings tracking
- **Membership Plans** — Free, Professional, Premium tiers
- **FAQ Accordion** — Interactive Q&A on membership page
- **Contact Form** — With validation and success toast
- **Newsletter Subscription** — Email capture in footer
- **Smooth Animations** — Scroll-based fade-in effects
- **Fully Responsive** — Mobile, tablet, and desktop layouts
- **SEO Optimized** — Meta tags on every page

## 🛠 Tech Stack

- HTML5
- CSS3 (Vanilla — no framework)
- JavaScript (Vanilla)
- Font Awesome 6 (Icons)
- Google Fonts (Inter + Playfair Display)

## 🚀 Deployment

1. Push the entire folder to a GitHub repository
2. Go to **Settings → Pages**
3. Set source to the `main` branch root (`/`)
4. Your site will be live at `https://yourusername.github.io/repo-name/`
